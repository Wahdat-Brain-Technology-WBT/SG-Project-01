# SHEEN GHAZY BABA ERP - Backend Architecture

This project implements a robust backend for a PVC/PPRC manufacturing company using FastAPI and PostgreSQL.

## Tech Stack
- **Framework:** FastAPI
- **Database:** PostgreSQL
- **ORM:** SQLAlchemy
- **Authentication:** JWT (JSON Web Tokens)
- **Validation:** Pydantic

## Project Structure
- `server.ts`: Main entry point (Express + Vite middleware for the environment)
- `backend/`: Python-based backend logic (to be implemented)
- `models.py`: SQLAlchemy database models
- `schemas.py`: Pydantic validation schemas
- `database.py`: PostgreSQL connection setup
- `auth.py`: JWT authentication logic
